Hosting and CORS Guide

Localhost

Run py -m http.server 8080 from this folder on Windows, then open http://localhost:8080/. Maximo must allow that origin.

Free static host

GitHub Pages, Cloudflare Pages, Netlify, and Vercel can host these files free. Add the published site origin to allowed CORS origins.

Proxy option

If CORS cannot be changed, place a controlled local or server-side proxy between the browser and Maximo. Do not expose secrets in public proxies.
